# Cindy
Sitio web moderno desarrollado con HTML, CSS y JavaScript para presentar el Combo Bienestar Exclusivo de Florece. Incluye un diseño responsivo, animaciones, modo claro y oscuro, secciones de beneficios, productos, ingredientes, testimonios, preguntas frecuentes y formulario de contacto, ofreciendo una experiencia atractiva y fácil de navegar.
